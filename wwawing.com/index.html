【32ビット版 Windows のサポート終了について】

32ビット版 Windows は Windows 10 の通常サポート終了により、ほとんどの場合利用が推奨されていません。WWA Wing の関連ツールでもサポートは終了しています。

WWA Wing の動作確認ツール (wwa-server.exe) の 2022年3月31日にリリースされた v3.7.0 以降のバージョンは、 32 ビット版 Windows では動作しません。

Windows 10 ESU などを利用している場合や、その他やむを得ない事情により32ビット版 Windows を利用する場合には、代わりに WWA Contest のサイトで配布されているWWA Debuggerのご利用を推奨します。

どこでも動く!

WWA Wingは、最近のほとんどのブラウザで動作します。 PCだけではなく、スマートフォンやタブレットでも動作します。

WWA Scriptという JavaScript ベースの独自言語で機能拡張ができます。また、 ピクチャ機能により、画像や位置、場所などを記載することで、ゲーム画面内に自由にピクチャを表示させることができます。
